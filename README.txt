bimtucklez.com static site (Under Construction)

Files:
- index.html / index.htm: Landing page (mobile-friendly).
- 404.html: Not-found page.
- robots.txt: Disallows /img/ from crawling.
- sitemap.xml: Basic sitemap for SEO.
- assets/css/styles.css: Styling.
- .htaccess: Disable directory indexes + caching (for Apache hosts).
- img/.htaccess: Blocks directory listing of /img.

Notes:
- On Cloudflare Pages, .htaccess is ignored (Pages does not use Apache).
- To add a background image, drop a file into /img and add CSS to body in styles.css.
